## [項目簡介](http://material.hocassian.com/prod-api/profile/video_matrix/intro.mp4)

在如今資本主導一切的市場大背景下，屬於我們的那份獨一無二，逐漸被他們製造出的“主流文化”所掩蓋。不知從何時??起，我們為了迎合“主流”，主動拋去了自己的特點，一頭扎進他們早先就設置好的社交圈中，企圖收穫更多的認同感；然後慢慢發現：只有那些將自己包裝得最精緻的人們，才能得到更多注目，獲得更多流量——於是我們開始了一輪又一輪的跟風，一邊為資本家們賣命，一邊又將被剝削過的工資還給他們換取所謂的“社會認同”以期自我實現……

在生物界，與復雜的自然環境一併產生的還有物種的多樣性，正是千千萬萬個生命之間的互相依存，才讓我們的星球有了生機和色彩。而對於擁有精神世界的我們來說，各異的文化、不同的思想、偶然的靈感……在它們的影響與碰撞下，百花齊放的文明誕生了，我們的生活更有意義了。而資本勢力企圖減少思想的維度，收束思考的層次，通過沒有硝煙的「和平演變」使拖尾效應不斷弱化，其結果就是——我們的情感逐漸失調，「精神溫室效應」不斷加劇，我們開始慢慢難以被打動，難以傾聽那共鳴之聲，我們的精神逐漸變得穩定、遲鈍、不再起伏……資本背後運轉的系統總是縝密而自洽的，我們不能期望從他們設計出的藍圖中突圍，最好的方式只有一個，退出這個遊戲，另起爐灶。

嘗試著將自己抽身出來吧，慢慢拾回那些曾經被遺棄的閃光點，漸漸回到出發的營地。你會發現，總會有那麼一群可愛的人們，在最初的起點等著你——這裡是未曾被資本侵蝕的小眾空間，這裡沒有攀比，沒有虛榮，沒有浮躁與喧囂，即便失去了物質的包裝，存在於自己心底的那個純淨的世界，也會在志同道合的伙伴們的鼓勵下不斷發光發熱，充盈自信心直至邁出腳步……於是乎，擦肩而過的我們，懷著相似卻不同的理想，走著相似而不同的道路，唱著同樣的一首歌，演奏著這個世界的交響。

多年以來，「同和系新媒體矩陣」致力於幫助亞文化小眾群體找尋自身價值，不斷扶持天才級創作者輸出優質內容，帶動相關領域持續細化，從中汲取養分使受眾圈煥發活力，成長為擁有自身擴張力的正能量模範群體。無論是音樂、繪畫、視頻亦或其他文化形式，我們皆有成功案例證明這種去中心化的運營模式能夠為用戶帶來正面的精神影響力，漸漸讓他們收穫彼此的認同和世界的善意，有動力尋找人生的寶物，點燃生命的亮光，最終擁抱美好的人生。

## 矩陣詳情

#### 同和視頻矩陣

在線體驗：https://hocassian.com

貯存&展示矩陣成員視頻稿件並藉此維護其原創性，提升矩陣影響力，防止公共平臺監管限流、無故下架、惡意帶節奏等信息霸權問題；促進成員相互引流&合作，以組織形式面向廣告主，實現共同盈利。

✅ 同步成員發佈在YouTube的視頻作品，集中展示與存檔備份；

❎ 根據視頻特徵信息不斷在全網檢測其版權狀態，若發現盜用，將第一時間通知原作者；

#### 同和素材矩陣

在線體驗：https://material.hocassian.com

旨在為創作者提供開源的高質多媒體素材，同時接收創作者的投稿與分享，提高多媒體素材的利用率；前接素材的採集，後接素材的調用與剪輯，為創作者省去查找素材的重複勞動。

✅ 成員之間共享圖片、音樂、視頻素材；

✅ 成員可用於整理歸檔自己的素材，提高創作效率；

✅ 借助AI智能為素材打標籤，實現精準分類，通過簡單的搜索就能找到想要的素材；

#### 同和創作矩陣

該矩陣旨在將現實/網絡/虛構三位一體，創建多層世界維度之間的連接&發展關係，為創作者打造全身心投入的抽象虛擬現實生態環境，這三個大脈絡圖譜又分別對應了不同的業務邏輯與需求。

✅ 通過六度人脈理論，構建領域大咖關係圖，幫助成員找到相關領域的合作夥伴；

✅ 以可視化的圖譜展示矩陣版圖，凸顯群體影響力；

❎ 跟隨社會熱點分享創意話題，相互提議激發靈感；

❎ 遇到版權糾紛不要慌，通過平台可一鍵咨詢法務；

❎ 共享商單信息，有飯一起恰；

#### 同和文學矩陣

文字作為傳播媒介之首，通之萬里而推之於百年；我們高度重視其深遠的領導力，並成立該矩陣，促進專欄作者與視頻創作者合作，將信息多元化，以期分類明確、高度集成和精緻包裝，順應未來傳播潮流。

✅ 同步成員在各大媒體平台發布的推文，集中展示與存檔備份；

❎ 根據文章特徵信息不斷在全網檢測其版權狀態，若發現盜用，將第一時間通知原作者；

#### 同和分發矩陣

我們致力於潛心研究媒體平臺，並最終獨創基於模擬器二次封裝的分發手段，且與多家分發軟件開發商達成合作，作者可一鍵將圖文或視頻一次性同步至多個自媒體平臺，真正意義上實現『一次分發，多端同步』。

❎ 分發主體文章、圖片、音樂、視頻作品到各大媒體平台，持續吸引不同平台相關領域的愛好者；

❎ 自動化分發降低重複勞動，使成員的注意力集中在創作輸出上；

## [創始人簡介](https://hocassian.cn/archives/other/7800)

退役Galgame編劇/b站優秀專欄作者，初中曾參與e7play旗下作品《夢想之翼》外包劇本策劃和《我們的回憶》劇情設計，曾獲得2012年全國青少年微電影劇本大賽TOP50；2015年個人Galgame劇本原案《暗影年華》獲得炒飯老師認可，並節選部分內容加入《高考戀愛100天》；2017年開始在知乎&b站&微信公眾號開設個人專欄，內容涉及影評、遊測、社論、IT等多個領域，至今已獲得100w+閱讀量。

b站&YouTube&西瓜視頻知名up主，精通拍攝、剪輯、特效、調色一條龍；2010年（初一）獲得人生中第一臺相機後就一直在記錄生活，至今積累了上千G的素材；並於同年拍攝的初中生活紀錄片獲得廣州市中小學信息設計大賽二等獎；高中擔任廣州六中校園DV社副社長一職，重啟《一周DV》計劃並開啟了b站的投稿生涯；2014年（高二）協助攝影社剪輯《冰釋舞中武》獲得2015年全國青少年微電影大賽最佳剪輯獎，2015年聯合社員拍攝《逃離》獲得2015年新概念微電影大賽最佳影片獎；2016年畢業季參與攝影社耽美短片《十八未滿》的製作，該影片一經發布，即刻在b站斬獲40w+播放的好成績；升上大學後加入深圳市秦桑影視有限公司在西江大學的駐點影視圈單位，拍攝《青春熱血》（因版權爭議在國內被下架）、《義膽拍檔》（因版權爭議被全網下架）、《深存記》等多部校園青春相關的微電影；2018年退出影視圈，轉戰動漫領域，加入騰訊·大粵網旗下『魔力TV』MCN成為微視app&企鵝號動漫簽約作者，而後經營自創企劃『同和系視頻矩陣』，並藉此參與第四屆中國“互聯網+”大學生創新創業大賽，榮獲銅獎&晉銀獎；2019年接受香港知名媒體『明報周刊』專訪，通稿亦刊登在國內知名二次元平臺『2次元現場』；2020年6月在國內知名青年導演夏夢怡劇情短片《聲音故事》的拍攝中擔任製片一職；個人原創視頻節目有Galgame測評相關的《戲說一下》、青年情感相關的《流年絮語》、時評社論相關的《翰哥怎麼看》（港澳臺地區別名《哈卡西亞的地平線》）、野外冒險相關的《玉笙&同和野採》、揭秘求真相關的《同和怪談記事簿》、記錄生活的vlog類視頻《同和日常》；以上節目在各大媒體平臺總共獲得了超過1000w次的點擊量，部分節目甚至爆紅海外，在港澳臺華人圈中皆有不錯的口碑。

2016年考入西江大學軟件學院進行本科知識的進修，2017年參與影視圈app相關接口的測試與調優，相關應用依托模型算法對數據進行全面分析測評，為廣大影視從業人員提供便利條件，並於同年3月份榮登2017北京電視節目交易會；2018年得益於影視圈創始人『陳霆聰』的人脈關係，工作室在華農設立新駐點，而我則趁此機會旁聽學習華南農業大學數字與新媒體技術專業的相關課程，並拿到相應的技能證書；同年11月開發基於 Python的智能雲分發系統，自動分發抖音&快手入駐達人的短視頻至其他各大自媒體平臺的對應賬號，免去用戶手動逐個上傳的繁瑣操作，同樣有效保護了用戶的版權，該項目已被知名IT媒體平臺“機器之心”收錄；2019年參與西江大學&東軟學院的合資實訓班，三個月內完全掌握SpringBoot&Vue.js&MySQL&RabbitMQ&微信小程序等技術棧的開發技能，緊接著加入先前合作過的傳媒單位『騰訊·大粵網（肇慶）』，運用IT技術開發項目『米果新媒體矩陣』切實服務地方媒體企業與私營新媒體公司，幫助他們在原有採編工作流不變的情況下快速完成面向互聯網的過渡與轉型；完成開發後即刻被公司投入實際生產，供下屬城市合作夥伴使用；2020年4月正式結束實習，6月大學本科畢業，並與7月入職字節跳動從事內部IT系統相關應用開發至今，包括位於行業前沿的會議效率工具「飛書妙記」的開發；此外我也經常開發一些方便創作工作流的小腳本（其中嗶哩嗶哩批量投稿腳本還被索尼音樂官方採納，上傳了十萬多個視頻），搭建了個人博客網站，且經營過一段時間的■■（數據刪除），DAU（日活）一度達到2000+；以及通過網絡工程破解■■■（數據刪除），為同學們提供更自由暢通的衝浪體驗。

網易音樂人/業餘攝影師/資深危機公關/NGO組織成員（曾）/中國左派青年，以上就是我的個人履歷了，人生有無數種可能，人生有無限的精彩，人生沒有盡頭，所以這份簡歷還會不斷更新。一個人只要足夠的愛自己，尊重自己內心的聲音，就算是真正的活著。
